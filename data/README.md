## Regenerating Data

### Raw Data (Script 01)
Requires internet access. Downloads from:
- **USGS NWIS** via `dataRetrieval::readNWISdv()` — Gauge 03068800, parameter 00060 (discharge)
- **gridMET** via `climateR::getGridMET()` — 8 climate variables, 1998-2023
- **MACA** via `climateR::getMACA()` — 10 climate variables × 2 RCP scenarios, 2024-2099

Runtime: Several hours (MACA downloads are large).

### Processed Data (Script 03)
Runs locally on raw CSVs. Applies stationarity corrections, standardization, and combines variables into model-ready datasets.

## CSV Column Descriptions

### Raw Climate CSVs
| Column | Description |
|--------|-------------|
| Year, Month, Day | Date components |
| Variable | Climate variable name |
| Mean | Spatial mean across watershed |
| Median | Spatial median |
| Max | Spatial maximum |
| Min | Spatial minimum |
| Range | Max - Min |
| SD | Spatial standard deviation |

### Combined Processed CSVs
| Column | Description |
|--------|-------------|
| Date | YYYY-MM-DD |
| Streamflow | Daily discharge in cfs (historic only) |
| Precip | Standardized precipitation |
| SolRad | Standardized solar radiation |
| MinTemp | Standardized minimum temperature |
| MaxTemp | Standardized maximum temperature |
| VPD | Standardized vapor pressure deficit |
